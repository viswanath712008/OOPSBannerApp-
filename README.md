# Java OOPS Program

This is a simple Java program that displays "OOPS" .

## How to Run

1. Ensure you have Java installed (JDK 8 or higher).
2. Compile the program: `oopsucs1.java`
3. Run the program: `oopsucs1.java`

## Output
```

```
